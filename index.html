About Me

Γεια σου! Είμαι ο πάνος, ένας παθιασμένος προγραμματιστής με εξειδίκευση στο Front-End Development και τη δημιουργία διαδραστικών εμπειριών.

Ασχολούμαι με την ανάπτυξη ιστοσελίδων και παιχνιδιών, χρησιμοποιώντας HTML, CSS, JavaScript, καθώς και άλλες σύγχρονες τεχνολογίες. Μου αρέσει να δημιουργώ όμορφα, λειτουργικά και εύχρηστα projects, είτε πρόκειται για ιστοσελίδες είτε για διαδραστικά παιχνίδια.

Έχω φτιάξει διάφορα projects, όπως:
- Ένα racing game με HTML, CSS και JavaScript 🚗💨
- Ένα horror game με jumpscares και σκοτεινά δωμάτια 👻
- Μια πλατφόρμα streaming για μουσική 🎵
- Ένα driving school simulator όπου ο παίκτης ακολουθεί τους κανόνες οδήγησης 🚦

Στον ελεύθερό μου χρόνο, μου αρέσει να εξερευνώ νέες τεχνολογίες, να βελτιώνω τα skills μου και να δουλεύω σε νέα projects. Αν θέλεις να συνεργαστούμε ή να μάθεις περισσότερα για τη δουλειά μου, μη διστάσεις να επικοινωνήσεις μαζί μου! 🚀
